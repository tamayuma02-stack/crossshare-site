🛡️ Windows の警告表示について（安全です）

CrossShare はデータを外部サーバーに送らない ローカル動作のアプリです。Microsoft の有料署名を付けていないため、初回起動時に「Windows によって PC が保護されました」という青い画面が表示されますが、ウイルスではありません。次の手順で起動できます。

  • ① 青い警告画面の 「詳細情報」 をクリック
  • ② 下に表示される 「実行」 ボタンをクリック
  • ③ 2 回目以降は警告は表示されません

よくある質問

購入前の疑問にお答えします。

Windows 版はインストールが必要ですか？

いいえ。インストールは不要です。ダウンロードした ZIP を解凍し、フォルダ内の crossshare.exe をダブルクリックするだけで起動します。フォルダごと好きな場所に置いて使えます。

Windows 版でプレミアムを購入できますか？

Windows 版では直接購入できません。プレミアムは Android 版（Google Play） で購入します。スマホで購入したあと、同じ Wi-Fi で Windows と接続すれば、接続している間 Windows でもプレミアム機能が使えます。

1台買えば他のデバイスもプレミアムになりますか？

はい。スマホで購入すれば、そのスマホと接続している全デバイスが、接続している間だけプレミアムになります。接続が切れると、購入していないデバイスは無料プランに戻ります。ただし同じ Wi-Fi に再接続すれば自動でプレミアムが再有効化されるため、再設定の手間はありません。（iOS・Mac 版の公開後は、各デバイスで個別に購入することもできます。）

インターネット接続は必要ですか？

デバイス間の共有には不要です。同じ Wi-Fi（ローカルネットワーク）につながっていれば、インターネットなしでファイルやクリップボードを共有できます。データが外部サーバーに送られることはありません。

無料プランでどこまで使えますか？

無料プランでも、2 台までの接続・クリップボード履歴 10 件・サイズ無制限のファイル転送・QR/PIN ペアリングが使えます。クリップボードの自動同期や接続台数無制限、広告非表示はプレミアム機能です。

データは安全ですか？

デバイス間の通信は暗号化されています。ペアリングは QR コードまたは PIN コードで認証し、保存されるデータはすべて端末内にのみ保存されます。詳しくはプライバシーポリシーをご覧ください。
